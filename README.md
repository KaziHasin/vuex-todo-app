# Todo App

A small app Todo listing using json placeholder fake api. I used Vue js and Vuex, axios library for building it.

## setup the app

clone or feth the app

cd vuex-todo-app

npm install

npm run dev

## API Reference

```http
 Methods => GET / POST/ PUT / DELETE
  https://jsonplaceholder.typicode.com/todos/
```

| Parameter | Type  
| `id` | `string` |
